### Route 1: Healthcheck
- **Path**: `/api/health`
- **Request Type**: `GET`
- **Purpose**: Verifies that the service is running and healthy.
- **Request Format**:
  - No parameters required.
- **Response Format**: JSON
  - **Success Response Example**:
    - Code: 200
    - Content: ```json {"status": "healthy"}```
- **Example Request**:
  ```bash
  GET /api/health
- **Example Response**:
  ````json
    {
      "status": "healthy"
    }
    ```
